# Sistema de gestion para Centros de Acopio - SOSVzla

Sistema en C para gestionar inventario de donaciones, con control de vencimientos, despachos, historial de movimientos y generación de árboles fractales en SVG para visualizar los productos más críticos.

## 📦 Requisitos
- Compilador C (gcc recomendado)
- Sistema operativo: Windows, Linux o macOS
- Navegador web para abrir los archivos SVG generados

## 🚀 Cómo compilar y ejecutar el codigo
```bash
gcc -o inventario main.c inventario.c -lm
./inventario
